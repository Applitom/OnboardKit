## Requirements

* Swift 4.0
* Xcode 9
* iOS 10.0+

## Installation

#### [Carthage](https://github.com/Carthage/Carthage)

````bash
github "NikolaKirev/OnboardKit"
````

Don't forget to `import OnboardKit` in the file you intend to use it.
